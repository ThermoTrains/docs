Übersicht

In diesem Kapitel wird die von uns entwickelte Vearbeitungspipeline erläutert. Der Input der Pipeline ist ein Video eines vorbeifahrenden Zuges. Das Endartefakt der Pipeline ist pro Zugwagon ein Bild welches den Wagon in einer nahezu orthografischen Projektion oder anders gesagt einer Planansicht darstellt. Ausserdem sollen unnötige Bildausschnitte entfernt werden wie Himmel oder Boden.
