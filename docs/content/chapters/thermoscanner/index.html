Thermo-Scanner

Die Thermobox verkörpert die Hard- und Software, um alle nötigen Rohdaten für die Analyse der Züge zu beschaffen. Sie soll am Gleisrand aufgestellt werden und alle vorbeifahrenden Züge automatisch aufnehmen. Die einzige Voraussetzung für den Betrieb ist ein Stromanschluss. Ansonsten soll die Thermobox autonom funktionieren.

Anforderungen

  • Die Box ist wetterfest.
  • Benötigt nur einen Stromanschluss, funktioniert nach dem Einstecken automatisch.
  • Die Aufnahme startet automatisch beim Einfahren eines Zuges.
  • Ein Infrarot- und ein Farbvideo wird aufgenommen
  • Metadaten zur Aufnahme werden aufgenommen (Kameraparameter, Temperatur, ...)
  • Die Wagonnummer wird mittels RFID ausgelesen um den Zug zu identifizieren
  • Die Vidoes und Metadaten werden auf einen Server hochgeladen.
  • Die Box kann remote gesteuert und gewartet werden.
  • Kann langsam fahrende Züge aufnehmen (Schritttempo).
